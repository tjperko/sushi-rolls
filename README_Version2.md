# Sushi Rolls Collectibles (static site)

This is a simple static single-file site (index.html) for the Sushi Rolls Collectibles demo.

Quick local preview:
- Open `index.html` in your browser, or run a local static server:
  - Python 3: `python -m http.server 8080` then visit `http://localhost:8080`
  - Node: `npx http-server -c-1` or `npx serve`
